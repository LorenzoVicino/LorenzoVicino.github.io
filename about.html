About

I like software that stays understandable.

I am a full-stack developer from southern Italy. I enjoy building products that solve practical problems without becoming difficult to maintain six months later.

Work

My day-to-day work is focused on web applications, product features, and backend logic that has to stay clear under real usage. I am especially interested in the line where product decisions, developer experience, and code quality meet.

Stack

Frontend: HTML, CSS, JavaScript, React, Redux.

Backend: Java, Spring, Node.js, REST APIs.

Data and tooling: PostgreSQL, MySQL, MongoDB, Docker, Git, Linux.

Approach

I prefer boring solutions when they are robust, and I like patterns only when they clarify the code instead of decorating it. Good software, for me, is software that another developer can enter quickly and change with confidence.

I also care a lot about interface restraint: fewer distractions, clearer copy, and pages that can be read without friction.

Outside Code

When I am not working, I usually split time between running, tennis, and reading about software architecture. Refactoring, naming, and system shape are things I keep returning to.
